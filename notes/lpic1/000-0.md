# Introduction

**Unix**: A powerful and widely-used operating system that originated in 1960 and has
since evolved into various flavors, including Linux.

**Unix Philosophy**: The philosophy that emphasizes on building a software with modular,
small, and reusable components.

**Linux**: Linux is an open-source, unix-like, and open-source operating system kernel,
developed by `Linus Torvalds` in 1991. It has been the heart of many different
open-source operating systems, commonly known as Linux distributions.

**Linux Distro**: A full and complete operating system, using Linux as its kernel. The
distribution include Linux system utils, libraries, and additional software.
