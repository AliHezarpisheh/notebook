# notebook

A personal space for structured learning.\
This repository contains my technical **notes**, **labs**, and **Jupyter notebooks**
across topics like Software Engineering, Linux, networking, DevOps, Data, and security.

## Structure

- **notes/** — Concept explanations, summaries, and references
- **labs/** — Hands-on exercises, configs, and troubleshooting steps
- **notebooks/** — Interactive Jupyter notebooks for experiments and demos

## Purpose

To build a consistent, organized workspace for deepening my understanding of computer
science.

> **❗By the way, the repo could be cloned and setup properly on your system, running the
> `./scripts/format.sh` to format the markdown texts you have.**
